LINUX

Uma Breve Introdução ao Mundo Linux

 LINUX é um kernel, o qual é comumente utilizado em servidores, porém, vem ganhando cada vez mais espaço nos desktops.
 Neste site, serão apresentados pontos importantes para quem ainda não teve contato com essa tecnologia, informando detalhes gerais, contando um pouco de sua história e no final haverá um Quiz para descobrir qual distribuição Linux se encaixa melhor com o seu perfil.
 Além disso, caso acabe se apaixonando pelo assunto, está disponível uma sessão de referências, a qual permite um acesso facilitado à conteúdos de qualidade sobre o assunto.

Por Que Usar Uma Distribuição Linux ?

  1. Estabilidade
  2. Portabilidade
  3. Liberdade
  4. Diversas opções e configurações
  5. Geralmente valorizam mais a privacidade do usuário
  6. Preço (as "distros" Linux geralmente são gratuitas)
  7. Melhor performance em computadores antigos
  8. Geralmente oferecem uma melhor experiência para desenvolvedores
  9. Distribuições como Ubuntu e Fedora disponibilizam ótimas "lojas de aplicativo", facilitando a instalação de programas.
  10. Ambiente UNIX

O Que é um Kernel ?

 Kernel é o núcleo de um sistema operacional, o qual gerencia os recursos do sistema e coordena os componentes. De forma simplificada, o Kernel é o "cérebro" do sistema operacional. Pelo fato de ser open source (ter seu código aberto), o Linux permite uma facilidade quanto à portabilidade, portanto, é encontrado em TVs Smart, microcomputadores e até celulares (o famoso Android utiliza o kernel Linux).
