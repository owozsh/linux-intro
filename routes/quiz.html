Fedora!

 O Fedora é uma distribuição desenvolvida pela Red Hat, uma empresa subsidiária da IBM, por padrão, essa distro entrega versões bem "cruas" das Desktop Environments, sendo as mais famosas o Gnome e o KDE Plasma. Além disso, trabalha com o sistema de atualizações constantes, permitindo ao usuário a utilização das ferramentas na sua versão mais atual.
 Geralmente, é uma distribuição que entrega diversas inovações, como o Fedora Silverblue, o novo sistema de arquivos Btrfs, entre outros. Por curiosidade, é a distribuição que o próprio Linus Torvalds utiliza, a qual entrega uma excelente experiência para desenvolvedores, especialmente os desenvolvedores Linux.
 Vale ressaltar que o Manjaro também é uma boa alternativa para o seu perfil, sendo as possíveis maiores diferenças a identidade visual apresentada e algumas configurações adicionais, porém, caso prefira a experiência pura, melhor ficar com o Fedora mesmo.

Arch Linux!

 O Arch Linux é uma distribuição que pode assustar à primeira vista, pelo fato de que instalação do sistema é bem mais manual em relação a outros Sistemas Operacionais.
 Apesar disso, o Arch é muito querido por sua comunidade, tanto por ter um dos maiores respositórios da comunidade, o AUR, quanto por estar repleto de possibilidades de customizações e outras possibilidades.
 Vale a pena dar uma olhada no Gentoo também.

Ubuntu!

 O Ubuntu é considerado uma das distribuições mais populares, possívelmente a mais utilizada entre os desktops Linux.
 O Ubuntu em si possui uma identidade visual muito forte, principalmente na época do Unity, e, por muito tempo, ditou algumas regras entre as distribuições.
 Atualmente o Ubuntu vem perdendo espaço para outras distros, mas, principalmente, pelas que foram baseadas no próprio, como o Pop OS e o Elementary OS, os quais, inclusive, talvez sejam uma melhor opção para os usuários que se encaixam com o seu perfil, portanto, vale a pena conferir!

Manjaro!

 Um Arch Linux simplificado, o Manjaro é uma distribuição extremamente rápida, ótima para jogos, tanto que, por padrão, ele já vem com a Steam pré-instalada. Para computadores mais antigos, o recomendado é utilizar a versão XFCE do Manjaro. Talvez seja válido dar uma olhada também no Xubuntu ou o Lubuntu.
